# react-demo
this is a react demo

index-jsx.html是用cdn加载react、react-dom、及babel实现对react 渲染、条件判断、列表、表单等的应用

# react-app 独立运行 
运用npm包的方式一个小demo，包含对antd的应用
进入文件夹 
```
npm install
npm run start 
```

# react-spa 独立运行
引入react-router-dom，实现单页，包含：
## react Hooks
- useState
- useEffect
- useContext
- useReducer

## router Hooks
- useParams
- useHistory

进入文件夹 
```
npm install
npm run start 
```
